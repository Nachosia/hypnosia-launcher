# Changelog

## [0.1.0] - 2026-06-17

### Добавлено
- Авторизация через Discord и по железу (HWID).
- Страница аккаунта с отображением ника, ID, роли и даты регистрации.
- Привязка Minecraft-аккаунта по коду.
- Автообновление лаунчера через GitHub Releases (Tauri Updater).
- Кнопка «Обновить» на странице аккаунта.

### Изменено
- `API_BASE` вынесен в переменную окружения `VITE_API_BASE_URL`.
- URL модов и зависимостей вынесены в переменные окружения Rust.
- Проект подготовлен к публикации в публичном репозитории.
