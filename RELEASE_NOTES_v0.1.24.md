# Hypnosia Launcher v0.1.24

## Новое

- **Восстановление аккаунта по ключу.** На экране входа появилась кнопка «Восстановить по ключу». Можно ввести 32-символьный ключ вручную или импортировать файл `account-{id}.properties`. Работает только после сброса HWID.

## Исправления

- **Привязка Discord теперь обновляет UI автоматически.** После подтверждения OAuth в браузере лаунчер поллит `/api/launcher/me` до тех пор, пока сервер не подтвердит привязку, и только тогда обновляет профиль.
- **Добавлено состояние загрузки и обработка ошибок** для кнопки «Привязать Discord». Если за 6 минут привязка не завершится, показывается ошибка timeout.

## Требования к backend

Эта версия лаунчера рассчитывает на обновлённый сайт (`tmp_launcher-router.ts` v0.1.24+):
- один Discord может быть привязан только к одному HWID;
- OAuth callback обновляет `mod_accounts.discord_id` и синхронизирует роль;
- исправлен `/api/launcher/link-minecraft`.
