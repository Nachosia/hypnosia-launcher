# Hypnosia Launcher

Клиент лаунчера для проекта Hypnosia / Nachosia.

## О проекте

Это десктоп-приложение для запуска Minecraft с кастомным набором модов и синхронизации аккаунта игрока. Собрано на Tauri 2 + React 19 + Vite 8.

## Почему часть данных скрыта

Репозиторий публичный, поэтому из исходников убраны все site-specific значения:

- адрес и эндпоинты backend/API;
- ссылки на загрузку модов и зависимостей;
- Discord / client secrets и любые ключи, дающие доступ к инфраструктуре проекта;
- приватный ключ подписи обновлений.

Причины:

1. **Безопасность.** Публичный код не должен содержать прямых ссылок на внутренние сервисы, чтобы исключить их злоупотребление, фишинг или атаки на инфраструктуру.
2. **Защита от подделок.** Если моды, API или ключи обновлений будут публично доступны, любой сможет собрать выдающийся за официальный лаунчер и распространять его от имени проекта.
3. **Контроль релизов.** Все чувствительные параметры встраиваются на этапе сборки через переменные окружения, доступные только команде проекта. Это позволяет выпускать обновления и менять backend-ссылки без риска утечки.

## Что остаётся публичным

- Общая архитектура приложения и UI.
- Публичный ключ проверки обновлений (он нужен клиентам, чтобы убедиться, что скачанное обновление действительно от команды проекта).

## Контакты / поддержка

По вопросам работы лаунчера обращайтесь к команде проекта через официальные каналы.
